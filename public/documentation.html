myFlix API

This is a server-side component of a movies web application. It will provide users with information about movies, genres, actors, etc. Users will also be able to sign up, update their personal information and create a list of favorite movies.

Request URL Request Parameter HTTP Method Response body data format Request body data format
Returns a list of all movies to the user "/movies" none GET
Example: app.get("/movies", (req,res) =>{})		 A JSON object holding data about all the movies
Example: {Title: "Moulin Rouge", Description: "", Director: { Name: "Baz Luhrmann", Bio: "" }, Genre: { Name: "Musical", Description:""} }
Returns data about a single movie by title to the user "/movies/:title" :title GET
Example: app.get("/movies/:title", (req,res) =>{})		 A JSON object holding data about a single movie
Example: {Title: "Moulin Rouge", Description: ""}
Returns data about a genre by name "/movies/genre/:name" :name GET
Example: app.get("/movies/genre/:name", (req,res) =>{})		 A JSON object holding data about a genre
Example: Genre: { Name: "Musical", Description:""}
Returns data about a director by name "/movies/directors/:name" :name GET
Example: app.get("/movies/directors/:name", (req,res) =>{})		 A JSON object holding data about a director
Example: Director: { Name: "Baz Luhrmann", Bio: "" },
Allows new users to register "/users" none POST
Example: app.post("/users", (req,res) =>{})		 A JSON object holding data about new user that was added
Example: { ID: Integer, Username: String, Password: String, Email: String, Birthday: Date }, 		A JSON object is requested
Example: { ID: Integer, Username: String, Password: String, Email: String, Birthday: Date },
Allows users to update their user info "/users/:username" :username PUT
Example: app.put("/users:username", (req,res) =>{})		 A JSON object holding the updated user data
Example: { ID: Integer, Username: String, Password: String, Email: String, Birthday: Date },
Allows users to add a movie to their list of favorites "/users/:username/movies/:movieID" :username,:movieID POST
Example: app.post("/users/:username/movies/:movieID", (req,res) =>{})		 A text message indicating that the movie has been added
Allows users to remove a movie from their list of favorites "/users/:username/movies/:movieID" :username,:movieID DELETE
Example: app.delete("/users/:username/movies/:movieID", (req,res) =>{})		 A text message indicating that the movie has been deleted
Allows existing users to deregister "/users/:username" :username DELETE
Example: app.delete("/users/:username", (req,res) =>{})		 A text message indicating that the user has been deleted
